java基础（无序版）（一）

发表于 2019-03-15 | 分类于 后端

前言

重新学习java，记录的东西可能比较杂乱。

标识符

只能以字母、下划线或$(美元符号)开始，后面可以接数字、字母、下划线或美元符号。

数据类型

在java中，数据类型分为两类：基本数据类型（primitive type）和引用类型（reference type）。

八种基本数据类型：

byte: 字节，占8位，取值范围-128~127

short: 短整型，占16位，取值范围-32768~32717

int：整形，占32位，取值范围-21474836~2147483647

long：长整形，占64位，取值范围-2^63~2^63-1

float:浮点型，占32位

double：双精度浮点型，占64位，存储带小数点的数字

char：字符型，占16位，存储单个字符，0-65535

boolean：布尔类型，占一个字节，用于判断真假（只有true和false），默认值为false。
