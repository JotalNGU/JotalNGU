java中的参数传递方式

发表于 2019-03-15 | 分类于 后端 java

前言

在程序设计语言中，参数传递有两种方式：值传递和引用传递。

值传递：表示传递的是调用者提供的值的拷贝。值传递中，对形式参数的进行修改不会影响实际参数。

引用传递：表示传递的是调用者提供的变量地址，也俗称“传地址”。对形式参数内容的修改会影响实际参数。

Java中只有值传递。也就是说，Java方法接收的都是实际参数的拷贝。

参数有两种类型：

  • 基本数据类型
  • 引用数据类型

下面将会从这两种类型通过实际的例子证明：Java是值传递。
