数组引用：

如果有数组a与b，执行a=b, a将指向b的内存地址，所以内容也相同。此时，修改任一数组中的值，另一数组的对应的值也会改变。

运算符

  1. 按位与运算：&

    化为二进制，按位运算，两位都为1，结果才为1；

  2. 按位或运算：|

    化为二进制，按位运算，两位都为0，结果才为0；

  3. 按位异或运算：^

    化为二进制，按位运算，两位相同为0，不同为1；

类和对象：

Java是一门面向对象的语言。

面向对象编程主要有三个特性：

封装性

面向对象编程的核心思想之一就是将数据和对数据的操作封装在一起。

继承

子类可以继承父类的属性和行为，子类也可以定义自己特有的属性和行为。

多态

多态有两种。一种是操作名称的多态，即多个操作有相同的名字，但是接受的信息类型不同。另一种是关于继承的多态，即同一操作被不同类型对象调用产生不同行为。

类

类是一种用于创建具体实例（对象）的数据类型。

类的创建：

类体

类声明之后的“{ }”以及他们之间的内容是类体，类体的内容由两部分组成：

  1. 变量的声明（属性）
  2. 方法的定义（对属性的操作）

成员变量

成员变量是在类体内，方法外定义。它的作用域在整个类之内都有效，与书写的先后顺序无关。

方法

方法由方法头和方法体组成，参数可有可无。

例如：

方法名前面的是方法的返回类型，如果是void就不需要返回。

{}以及里面的内容是方法体。

方法体定义的变量是局部变量。局部变量只在方法内有效，而且与声明的位置有关。方法的参数在整个方法内有效。局部变量在其声明位置之后才有效。如果是复合语句内（如for循环）声明的变量。只在该复合语句之内有效。

注意：如果局部变量和成员变量同名，那么成员变量会被隐藏。若要使用隐藏的成员变量，必须使用this配合使用。

注意：成员变量有默认组，局部变量没有默认值。因此，在使用局部变量的时候要格外注意局部变量的赋值。

构造方法

构造方法是类中一种特殊的方法，当程序用类创建对象的时候会用到构造方法。特性如下：

  1. 名字和类名完全一样，并且是 void类型。
  2. 允许在类中定义多个构造方法，但是参数必须不同：参数个数不同或参数类型不同。
  3. 如果类中没有自定义构造方法，系统会有一个默认的构造方法（无参数）。
  4. 在创建对象的时候自动调用。

ps:如果自定了构造方法，系统不再提供默认的构造方法。
