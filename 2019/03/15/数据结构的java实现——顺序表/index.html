数据结构的java实现——顺序表

发表于 2019-03-15 | 分类于 后端 数据结构 java

前言

在数据结构中，线性表是一种简单而常用的数据结构，也是其他复杂数据逻辑结构的基础。线性表中数据元素是“一个接一个”依次排序的，排在第一的数据元素称为表头，最后的数据元素称为表尾，在某个元素的前一个称为该元素的直接前驱，后一个称为直接后继。对于非空线性表来说，表头是唯一一个没有直接前驱的数据元素，表尾是唯一一个没有直接后继的元素。同一个线性表中，数据元素需要同一个数据类型。

线性表有两种存储结构：

  • 顺序存储
  • 链式存储

顺序存储，是使用一组地址连续的存储单元一次存放线性表中各个元素。它是顺序存储，随机存取。

顺序存储，在计算机中常常用数组的形式进行存储。所以，接下来，我会用java语言的数组来实现顺序表的操作。
